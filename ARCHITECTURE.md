# Voice Clone Project Architecture

## Цель
Zero-Shot клонирование голоса на базе F5-TTS. Локальный инференс без дообучения.

## Железо и ограничения
- GPU: NVIDIA RTX 3060 12GB VRAM
- RAM: 32GB
- OS: Windows
- Лимит VRAM для TTS: ~8-9GB (остаток под систему/Ollama)
- Генерация: ~3-5 сек речи за 1 сек реального времени

## Структура
- voices/ — эталонные WAV (24kHz mono, 30-60 сек чистой речи)
- outputs/ — результаты синтеза
- src/config.py — настройки путей и параметров
- src/audio_utils.py — загрузка, нормализация, сегментация, склейка
- src/tts_engine.py — обёртка над F5-TTS API
- src/api.py — FastAPI endpoints
- main.py — точка входа

## Технические требования
1. Входное аудио: только WAV PCM 24kHz mono. Нормализация до -20dBFS при загрузке.
2. Текст: сегментация по 200-300 символов на границах предложений. Кроссфейд 15мс при склейке.
3. Кэширование: эталоны хранить в RAM (не VRAM). Модель загружается один раз.
4. Потоковая генерация для длинных текстов.
5. Без предобработки текста на первом этапе (опционально позже через Qwen).

## Запрещено
- Использовать глобальный Python или aider-env
- Загружать эталоны в VRAM
- Генерировать текст >300 символов одним батчем
- Устанавливать лишние зависимости без согласования
